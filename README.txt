====================================================================================================

Programa que usa la base de datos ORACLE para simular una bibloteca de juegos a partir de un archivo
.csv el cual servira como "tienda". Existe la posibilidad de que el programa no funcione 
correctamente pero eso se debe netamente a una mala conexion entre el lenguaje Python y la base de 
datos, lo cual como usuario debera revisar antes de comenzar el programa.

====================================================================================================

En la consola abrir el trabajo en la ubicacion del archivo .py con el siguiente comando:
	> py tarea.py

Inmediatamente comenzara a cargar los datos del csv a la base de datos de ORACLE.
Unicamente se requiere que use la consola, no es necesario abrir nada mas para el funcionamiento del
programa.
